# Module-6-Data-Structures-and-Strings-in-Python

# Python Data Structures Tasks

## Task 1: Student Marks Dictionary
This program stores student names and marks in a dictionary.  
It allows the user to enter a student name and retrieves the marks.  
If the student does not exist, an appropriate message is displayed.

## Task 2: List Slicing
This program creates a list from 1 to 10, extracts the first five elements,
reverses them, and prints both the original extracted list and the reversed list.

- Python 3
